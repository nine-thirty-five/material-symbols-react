# Material Symbols React — Showcase

An interactive playground for the package: switch style, weight, fill, size and color, search the
full icon set by name/tag/category, copy the exact import for any selection, and view any icon
across all 42 variants at once.

## Run it

The example resolves the package from the local build, so generate a `dist` first (from the repo
root):

```sh
# repo root — a quick sample is enough, or `npm run build` for all icons
npm run generate:sample
```

Then, in this folder:

```sh
npm install
npm start
```

Open [http://localhost:1234](http://localhost:1234).
